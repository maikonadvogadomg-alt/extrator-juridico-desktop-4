TRANSFORMAR EM APK (sem instalar Android Studio)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  1. Crie conta GRÁTIS em  github.com
  2. Crie repositório público em  github.com/new
  3. Instale o Git em  git-scm.com
  4. Abra o terminal nesta pasta e execute:

       git init
       git add .
       git commit -m "Extrator Juridico"
       git remote add origin https://github.com/SEU-USUARIO/SEU-REPO.git
       git push -u origin main

  5. Aguarde ~10 min → clique em Releases → baixe o .apk
  6. Transfira para o celular e instale

  Detalhe completo: veja MANUAL-COMPLETO.html (seção 6)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
