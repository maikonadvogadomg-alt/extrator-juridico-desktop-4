  • WebVM    — Alpine Linux via WebAssembly (sem servidor, offline)

  • CoCalc   — Terminal Linux na nuvem

GERAR APK (aba APK Builder)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  OPÇÃO A — Capacitor + Android Studio (APK local):
    1. Arraste o ZIP do seu site
    2. Preencha Nome e Package ID (ex: br.adv.maikoncaldeira.app)
    3. Escolha orientação (retrato/paisagem/ambas)
    4. Clique "Gerar ZIP do Projeto"
    5. Extraia o ZIP → npm install → npx cap copy android
    6. Abra Android Studio → pasta android/ → Build APK

  OPÇÃO B — GitHub Actions (sem Android Studio):
    1. Mesmos passos 1-4 acima
    2. Suba o ZIP no GitHub
    3. APK gerado automaticamente — baixe em Releases

  OPÇÃO C — Electron (executável PC):
    1. Mesmos passos 1-4 com destino "Electron"
    2. npm install && npm start   (para rodar)
    3. npm run dist               (para gerar .exe/.dmg/.AppImage)

  GERAR APK DO PRÓPRIO IARA SUITE:
    1. Clique "Demo (Iara Suite)" no APK Builder
    2. Escolha orientação "Ambas"
    3. Gere o ZIP → compile normalmente
