7. Site Extractor

O Site Extractor acessa páginas web via proxies públicos gratuitos para contornar restrições de CORS, permitindo analisar scripts de cálculo de sites jurídicos.

Proxies Públicos Utilizados

  1. allorigins.win — proxy primário, retorna JSON com HTML
  2. corsproxy.io — fallback automático se o primeiro falhar
  3. codetabs.com/v1/proxy — terceiro fallback
Sites com Cloudflare Enterprise ou WAF avançado podem bloquear proxies. Nesse caso, experimente acessar o site e inspecionar manualmente o HTML via F12 → Fontes.

Scripts de Cálculo Detectados

O extrator identifica automaticamente scripts com:

  • ⚖️ Cálculo — palavras como: salário, benefício, INSS, previdenciário, aposentadoria, pensão, correção monetária, coeficiente, RMC
  • 📐 Fórmula — Math., parseFloat, toFixed, function calc, operações matemáticas
  • 🌐 AJAX/Fetch — fetch(), XMLHttpRequest, $.ajax(), axios

Funcionalidades

  • ⬇ .js — baixa todos os scripts inline em um arquivo JavaScript
  • ⬇ HTML — baixa o HTML completo da página
  • 👁 Ver — abre o código do script em modal com opção de copiar
  • 🔗 — abre o arquivo externo em nova aba
  • Profundidade 2 — extrai também subpáginas (até 5 links)

9. FAQ — Perguntas Frequentes

O app usa o servidor do Replit de alguma forma?

Não. Nenhuma chamada vai ao Replit. A IA chama diretamente o servidor da provedora (OpenAI, Groq, etc). O terminal usa WebVM (WebAssembly local). O extrator usa proxies públicos independentes.

Minha chave de API fica segura?

A chave é salva apenas no localStorage do seu navegador (no seu dispositivo). Ela nunca é enviada a nenhum servidor intermediário — vai direto da sua máquina para o servidor do provedor (OpenAI, Groq, etc).

O APK Builder gera o APK automaticamente?

O APK Builder gera o projeto de código-fonte em ZIP. Para gerar o arquivo .apk instalável, use o Android Studio (local) ou o GitHub Actions (automático na nuvem). O processo via GitHub Actions é mais simples — sem instalar nada.

Posso converter o Iara Suite em APK?

Sim! Na aba APK Builder, clique Demo (Iara Suite). Isso carrega o próprio app com configuração br.adv.maikoncaldeira.iarasuite e orientação "Ambas". Gere o ZIP e compile normalmente.

O Terminal Linux funciona offline?

O WebVM roda Alpine Linux via WebAssembly. Ele precisa de conexão inicial para carregar (~10 MB), mas depois funciona parcialmente offline. Para acesso à internet dentro do VM, configure o Tailscale (opcional).

O Site Extractor não conseguiu acessar um site. O que fazer?

Sites com proteção Cloudflare podem bloquear proxies. Opções: (1) tente a URL sem www; (2) use a profundidade 1; (3) acesse o site manualmente no navegador e salve a página (Ctrl+S), depois extraia o HTML salvo.

Posso usar múltiplas IAs simultaneamente?

Configure os 4 slots com chaves diferentes. Apenas um slot pode estar ativo por vez. Para trocar, abra 🔑 Chaves API, ative outro slot e salve.
